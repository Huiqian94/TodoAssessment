Instructions to run TODO assessment (Cognixus): 

1. Open terminal. 
2. Run this commands accordingly: 
   
  a. git clone https://github.com/Huiqian94/TodoAssessment.git  
  b. cd {THE CLONED PROJECT DIRECTORY}/TodoAssessment
  c. docker-compose up
  d. Leave the terminal idle after build complete. 
  
3. Open Postman. 
4. Import the following file into Postman:  
  "TodoAssessment Apis.postman_collection.json" 
5. Run any api as required. 

*note: for first run, Postman might encounter a timeout - please rerun the api.
